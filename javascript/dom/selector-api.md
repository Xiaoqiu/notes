
## 选择器
### `document`
- `document.getElementById()`: 通过元素Id
- `document.getElementsByTagName()`: 通过标签名称
- `document.getElementsByName()`: 通过元素的Name属性的值
- `document.getElementByClassName()`: 通过类名

### `element`
- `element.getElementsByTagName()`
- `element.getElementsByClassName()`

### Selector API
- `querySelector()`
- `querySelectorAll()`
