# Module

- `CommonJS`: Node.js; Browserify
- `AMD` ( Asynchronous Module Definition ) : RequireJS
- `CMD` ( Common Module Definition ) : SeaJS
- `UMD` ( Universal Module Definition ): 同时兼容AMD和CommonJS
- `ES6 Module`
