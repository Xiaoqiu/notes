# `box-sizing`

```css
/*
 * 默认值，width和height属性不受padding和border影响
 * 实际页面中元素的实际宽度是4者相加的结果
 */
box-sizing: content-box;

/*
 * width和height属性包括了padding和border
 * padding和border的改变不会对实际宽度造成影响
 */
box-sizing: border-box;
```
