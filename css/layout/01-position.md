# position

```css
position: static; /* 默认值 */
position: relative; /* 相对定位 */
position: absolute; /* 相对于position值不是static的最近祖先元素（或者body）定位 */
position: fixed; /* 相对viewport定位 */
position: sticky; /* 粘性定位, relative和fixed混合效果 */
```

## sticky
仅支持Safari和Firefox
