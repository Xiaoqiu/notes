# devtools

## console
- console.log()
- console.info()
- console.warn()
- console.error()
- console.dir()
- console.count()
- console.time(label)
- console.timeEnd(label)

## selector alias
- `$(selector)`: `document.querySelector()` 的别名
- `$$(selector)`: `document.querySelectorAll()` 的别名
