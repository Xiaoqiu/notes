## 5. 简述CSS的两种加载方式：link和@import的区别
- 使用 `link` 标签引入的CSS文件，浏览器会并行下载
- 使用 `@import` 语法引入的CSS文件，会在页面所有资源加载完毕后下载
- CSS优先级与下载顺序无关
