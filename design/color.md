# Color

## 加色
### RGB
- R: Red 0~255
- G: Green 0~255
- B: Blue 0~255

### HSL
- H: Hue 色相
- S: Saturation 饱和度
- L: Lightness 明度

### HSB/HSV
- H: Hue 色相: 范围0~360
- S: Saturation 饱和度: 色彩的纯度, 越高色彩越纯, 低则逐渐变灰, 取0-100%
- B/V: Brightness/Value 亮度: 物体表面的反光率, 取0-100%, 0为黑
